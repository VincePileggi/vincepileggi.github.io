# Evidence-Based Engineering

A public educational resource dedicated to environmental engineering, applied statistics, and reproducible technical reporting.

## Overview

This website presents engineering frameworks, statistical methods, and analytical workflows intended for educational and professional development. Topics include:

- Wastewater treatment performance assessment
- Engineering statistics and uncertainty analysis
- Environmental data analytics
- Reproducible reporting using Quarto
- Python and R for environmental engineering
- Engineering education and mentoring
- Technical communication in plain language

## Website

Visit the website here:

https://vincepileggi.github.io

## Guiding Principles

- Teach methods, not projects.
- Use evidence before opinion.
- Communicate uncertainty clearly.
- Prefer reproducible workflows.
- Keep examples generalized, synthetic, or based on public information.
- Avoid disclosing confidential, proprietary, or project-specific information.
